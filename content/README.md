# Content Directory

存放所有 Markdown 内容文件。

## 目录结构

- `finance/` - 金融投资
- `travel/` - 旅行攻略  
- `ai/` - 人工智能
- `work/` - 工作职场
- `thinking/` - 思考随笔
- `reading/` - 阅读笔记
- `resource/` - 资源工具
- `life/` - 生活日常

## 文件格式

每个 .md 文件需要包含 YAML frontmatter：

```yaml
---
title: "文章标题"
date: "2026-02-15"
source: "https://original-url.com"
category: "finance"
tags: ["article", "investment"]
status: "unread"
author: "作者"
summary: "一句话总结"
---
```
